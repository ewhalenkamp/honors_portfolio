Project Description

This project lists each and every Pokémon instance until the start of the current Generation, Gen VIII. By clicking on a Pokémon's image, the site user can view individual and detailed information about that Pokémon, including their stats, moves, abilities, and more.

This Pokédex succeeds a Pokédex I created for a class project. As it was an introductory programming course and I didn't know how to use modern web technologies at the time, its initial implementation was completed by compiling nearly a thousand HTML files using a Java program.

To update this project, I refactored my Pokémon Java database to accompany a Springboot API, and then created a Django backend to serve the HTML with API-provided context instead of hard-written values. This brought down the number of total HTML files from over a thousand to just two.

This was my first experience creating an API, and I was able to apply this to future proprietary APIs (namely, RecordWall). Projects hosted on Heroku's free version are slow to initially load, which obviously isn't ideal for an API. Nonetheless, this fact lead to optimization after optimization until the long list page loaded in a reasonable amount of time.
