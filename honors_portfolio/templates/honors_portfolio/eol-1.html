Project Description

RecordWall is an app that allows for real-time sharing of music with all your friends. With RecordWall, you can start a room (wall) where everyone you invite is able to add their music to a public queue. Filter which songs you wouldn't like to hear for a more streamlined music sharing experience!

RecordWall is currently only available on the App Store for iOS devices. Includes user accounts/verification, third party and proprietary APIs, in-app purchases, advertisements, and a growing database of licensed music using PostgreSQL and AWS S3.
