Project Description

Easel Gradebook is a Web Application that allows users to experience a course organization system similar to Blackboard, Google Classroom, and Canvas, for which it is affectionately named.

Users are able to create classes for which they are the new instructor. There, they may create assignments, manage the course discussion board, and grade work submitted by students, other users that join the course after its creation.
